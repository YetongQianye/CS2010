项目简介
		背景：随着电子及网络技术的迅速发展，我们对于传统的纸质日历或挂历已经不再满足
			 而精致的电子日历或许能够吸引人们的眼光
		
		简单介绍：本项目主要利用虚拟机Ubuntu系统及ARM开发板进行项目开发工作，通过在开发板的LCD屏幕上显示时间日期及图片
				
		思路：首先通过取字模及时间换算方法，然后延时，在屏幕上显示时间、星期、年月日；然后利用链表存储图片，lcd显示出来；
			 然后通过触摸屏幕实现图片的前后切换及自动播放。
		
		用到的知识点：lcd屏幕的文字及图片显示，lcd屏幕触摸，多进程
		
		难点以及解决方法：首先碰到的是在将时间日期显示模块放到图片显示上时，两者不能共存的问题。经过向同学请教后用了多进程
				方法，把时间日期显示模块和图片显示的函数分别用一个进程进行，然后就可以同时显示。然后又碰到了文字
				被图片遮挡的问题，经排查后通过先画方块，然后字体在方块上面显示解决。
